### Framework
* #### Four all features:
  *    **Multithreading** - process the image pixels in parallel.  
  *    **Antialiasing** by generating multiple primary rays per pixel and averaging.
  *    **Add Box** 
  *    **Motion blur** - have a moving object, and average the result of multiple renders.
